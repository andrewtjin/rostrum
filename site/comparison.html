How Rostrum compares

Verbatim and most in-round tools are VBA .dotm macros. Rostrum is a modern Word add-in. That difference is not cosmetic — here is what each tool buys you, tool by tool.

At a glance

  Rostrum (Office.js add-in) Verbatim & VBA macros
Time to hide a real tournament case file ~4–5 seconds — measured on real 35- and 49-card case files ~5–10 minutes on the same files
While it works through a long document Word stays responsive — switch to your other open documents and keep working Runs on Word's UI thread, so the whole app is frozen until it finishes
Reversing shrink / condense Lossless — rebuilt from markers inside the document itself Not guaranteed to reconstruct from the document alone
Opening a processed file on another machine The same view re-derives from data inside the file Depends on the recipient having the template installed

Below, each Rostrum ribbon tool against the Verbatim tool it replaces — what you actually gain by switching.

Invisibility Mode

Hide and Show All. The job matches Verbatim's invisibility — hide everything that isn't a tag, cite, or highlight — but Rostrum wins on the things you feel mid-round.

Word never freezes

Rostrum runs in its own process, so a hide never locks Word — start one on a huge case file and immediately switch to your other open documents and keep prepping while it runs. A Verbatim macro runs on Word's own UI thread, so the whole app is frozen until it finishes. Even a 900k-word file stays responsive the entire time.

A cleaner hidden page

Rostrum's hidden text closes up to the left into a dense, straight column you can actually skim. Verbatim leaves the spacing of the hidden words behind, so your highlights sit scattered at their old positions with ragged gaps — and blank rows where whole lines were hidden. Same cards, far less visual noise.

Re-hides the same way, on any machine

Rostrum writes the armed state and your keep-colors into the file itself, so after you edit and press Hide again it re-derives the same view deterministically — on any machine, with nothing to set up. Verbatim's invisibility is a manual on/off toggle with no saved state to re-derive from.

Condense & Shrink

Shrink, Condense, and their reversals. Rostrum's edge here comes down to one word: lossless.

Reverse it from the document alone

Uncondense and Unshrink rebuild the original from self-describing markers inside the file — no sidecar, no add-in needed to reverse, and nothing extra shows up in your text. Verbatim's Condense is reversible only in its pilcrow mode (which drops literal ¶ marks into the running text); its default merge is one-way — the original breaks are gone for good.

Keeps more of the card full-size

Rostrum's Shrink keeps the underlined cut plus your highlights, cites, and headings at full size; only the surrounding analysis steps down. Verbatim's Shrink keys on underline alone, so anything you highlighted but didn't underline shrinks with everything else.
