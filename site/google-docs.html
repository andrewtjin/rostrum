Rostrum for Google Docs

Invisibility Mode & debate styles in a Google Doc — no Marketplace install.

You paste one script into the doc's own Apps Script project; a Rostrum menu appears after a reload. About two minutes.

Version v__GDOCS_VERSION__ · this is an early MVP (Hide / Show All, Apply debate styles, Mark cite). Full usage guide: gdocs/README.

What "Hide" actually does (honest posture). Hide shrinks every word of body text except headings, cites, and highlighted text down to a 1‑point size, so a long file reads like a speech doc and the page count collapses. It is not secrecy — anyone can reveal it with Select All → font size 11 (see the escape hatch below). Show All brings every word back. This matches how Rostrum behaves in Word.

2 · Paste it in

You install per‑doc, into the document's container‑bound script. Nothing is published anywhere; the script lives inside the doc.

  1. Open the Google Doc you want Rostrum in.
  2. Extensions → Apps Script. A new editor tab opens with an empty Code.gs.
  3. Select everything in Code.gs and delete it, then paste the entire contents of the downloaded Code.gs in its place.
  4. Tell the project which permissions it needs (one‑time manifest step): click Project Settings (the gear, left rail) and tick "Show 'appsscript.json' manifest file in editor." Back in the Editor, open appsscript.json and replace its contents with the downloaded one. (Alternatively, just add the Docs API advanced service: Editor → Services (＋) → Google Docs API → Add.)
  5. Save (Ctrl/⌘+S).
  6. Reload the Google Doc tab. After the reload a Rostrum menu appears in the menu bar (next to Help).

✓ A Rostrum menu now sits in the doc's menu bar.

3 · First run (the "click again" quirk)

The first time you pick a Rostrum action, Google interrupts to ask for permission — your click is consumed by the consent flow, not the action.

  1. Pick Rostrum → Hide (or any verb). A Google authorization dialog appears.
  2. Choose your account.
  3. Because this is your own unverified script, Google shows a warning screen. Click Advanced, then "Go to (your doc) (unsafe)." (Expected for a personal script you pasted yourself — you're granting it to your own doc.)
  4. Review the permissions and click Allow.
  5. The dialog closes without running the action — this is normal. Click the same Rostrum action again and it runs.

Rostrum runs as the account you authorized — the one signed into this Chrome profile when you click Allow. On a managed / school (Workspace Education) account, admin policy may block consent entirely; if so, install on a personal Gmail account instead.

Escape hatch — reading a hidden doc without Rostrum

If you (or anyone you send the doc to) ever need the text back without Rostrum:

Select All (Ctrl/⌘+A), then set the font size to 11. Everything becomes visible again.

Sharing with teammates

A Google Doc's bound script travels with a copy. To give a teammate the styled, Rostrum‑enabled doc: File → Make a copy (or share a copy). The copy carries the bound script, so they get the Rostrum menu after their own first‑run authorization. There is no central install to manage. (These shared copies aren't counted as downloads — see Privacy.)

Team norm: don't edit a doc while it's hidden — Show All first. Moving or copying hidden text can separate it from its saved sizes; Show All still brings it back (with a note), but editing un‑hidden is cleaner.

Limits in this version
  • Single‑tab docs only. A doc using Google Docs tabs is refused cleanly.
  • Suggesting mode: Hide and Apply debate styles refuse while there are unresolved suggestions; Show All always works.
  • Cut/paste of hidden text can detach it from its saved sizes; Show All then normalizes it back to full size and tells you how many passages it reset.
  • Condense/Shrink and Marketplace publication are not part of this version.
No tracking inside Google Docs. The pasted script only ever changes font size and named‑style definitions; it sends nothing back as you use it — no document content, no personal data, no usage pings. The single anonymous number Rostrum counts is the one‑time Code.gs download from this site (and only via the counted link above, not the direct copy). Full disclosure: Privacy.
