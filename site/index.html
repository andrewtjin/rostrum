Rostrum

A modern debate tool suite, built for debaters by debaters.

Collapse a debate doc to a headings, cites & highlights view and format your cards (reversibly) in the two editors debaters actually use. Pick yours:

Microsoft Word (v__ROSTRUM_VERSION__) Windows & Mac (desktop) Register one manifest.xml in Word, once. Then the Rostrum ribbon rides every document. Install for Word → Google Docs (v__GDOCS_VERSION__) Google Apps Script (browser) Make your own copy of a ready-made template; the Rostrum menu travels with the copy. Each document is its own copy, so grab a fresh one when a new version ships. About a minute, no Marketplace. Install for Google Docs →

The same idea on both

  • Hide collapses every body run except the keepers (headings, cites, and highlighted text) into a dense, skimmable view; Show All brings every word back.
  • Reversible without Rostrum. In Word, hidden text is Word's own Hidden font attribute. In Google Docs, Hide is a font-size shrink (not secrecy) that anyone undoes with Select All → font size 11.
  • Debate styles. Pocket / Hat / Block / Tag / Normal, one convention across both editors.
Two engines under one name. The Word add-in and the Google Docs port are built separately for each editor's document model, so they behave a little differently and update on their own schedules, but the keep rules and the “Show All brings everything back” promise are identical. The speed benchmarks you'll see are measured on Word; the Google Docs version is an early MVP.
